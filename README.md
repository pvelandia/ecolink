# EcoLink

Plataforma web para la gestión e incentivación del reciclaje mediante un sistema de puntos y convenios empresariales.

## Descripción del proyecto

EcoLink fue desarrollado como proyecto académico de Ingeniería de Sistemas en la Universidad de Cundinamarca.

El sistema busca promover prácticas de reciclaje en los hogares mediante un esquema de recompensas, donde los usuarios acumulan puntos que posteriormente pueden canjear por beneficios ofrecidos por empresas aliadas.

## Problema abordado

La baja participación ciudadana en programas de reciclaje y la falta de incentivos para la correcta disposición de residuos.

## Objetivos

- Fomentar el reciclaje.
- Incentivar la participación ciudadana.
- Conectar hogares con empresas aliadas.
- Gestionar el sistema de recompensas.

## Tecnologías utilizadas

- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
